2. QUE NO SE DEBE PUBLICAR

No publicar en carpetas publicas:
- Cedulas
- Fichas medicas
- RSH
- Autorizaciones notariales
- Planillas privadas con datos personales
- Comprobantes personales
- Archivos con RUT, telefono o correo de familias

Esos datos deben quedar solo en:
- Centro Operativo privado
- Carpetas Drive privadas del proyecto
- Apps Script/Google Sheets bajo control del club


3. OPCIONES PARA PUBLICAR

Opcion recomendada para empezar:
Netlify Drop

Ventajas:
- Arrastrar carpeta y publicar.
- Entrega enlace web real.
- Funciona en celulares.
- No requiere servidor propio.

Otras opciones:
- GitHub Pages
- Firebase Hosting
- Hosting del club si existe


4. PASOS PARA PUBLICAR EN NETLIFY DROP

1. Crear una copia limpia de la carpeta APP_PUBLICA_GIGANTES_AZAPA.
2. Revisar que config.js tenga la URL correcta de Apps Script.
3. Eliminar archivos internos que no deben publicarse si corresponde:
   - COPIAR_APPS_SCRIPT_CARGA_DOCUMENTOS.html
   - COPIAR_CODIGO_APPS_SCRIPT.html
   - APPS_SCRIPT_GIGANTES_WEBAPP.gs
   - diagnostico_avance.html
   - pruebas.html
4. Mantener revision_directiva.html solo si se quiere compartir con directiva.
5. Subir la carpeta a Netlify Drop.
6. Probar en celular.
7. Recién despues compartir enlace.
